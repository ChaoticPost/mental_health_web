О системе мониторинга психического здоровья

Описание проекта

Система мониторинга психического здоровья по текстовым сообщениям разработана в рамках ВКР по цифровой кафедре "Разработка и тестирование сервисов искусственного интеллекта". Система позволяет анализировать текстовые сообщения для выявления потенциальных проблем с психическим здоровьем, оценивать уровень риска и предоставлять соответствующие рекомендации.

Система использует методы обработки естественного языка (NLP) и машинного обучения для анализа текста, а также включает компоненты для анализа настроения и эмоций на основе современных трансформерных моделей.

Функциональность системы

Анализ текстовых сообщений

Система анализирует текстовые сообщения для выявления признаков проблем с психическим здоровьем, определяет категорию проблемы и оценивает уровень риска.

Анализ настроения и эмоций

Система определяет общее настроение сообщения (позитивное/негативное) и анализирует присутствие различных эмоций в тексте.

Оценка рисков

На основе анализа текста система определяет уровень риска для психического здоровья (низкий, средний, высокий) и формирует соответствующие рекомендации.

Персонализированные рекомендации

Система генерирует персонализированные рекомендации в зависимости от выявленных проблем и уровня риска.

Технологии

Система разработана с использованием следующих технологий:

  • Python - основной язык программирования
  • Flask - веб-фреймворк для создания API и веб-интерфейса
  • NLTK - библиотека для обработки естественного языка
  • Scikit-learn - библиотека для машинного обучения
  • Chart.js - библиотека для визуализации данных
  • Bootstrap - фреймворк для создания адаптивного веб-интерфейса

Важное примечание

Система не является заменой профессиональной помощи. Если вы испытываете серьезные проблемы с психическим здоровьем, обратитесь к квалифицированному специалисту.
